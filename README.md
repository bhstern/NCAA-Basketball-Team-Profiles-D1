# Benj Stern — CBB Team Profiles
## Setup Instructions

### Folder Structure Required
```
cbb_site/
├── index.html
├── convert_data.py
├── data/
│   └── cbb_data.json        ← generated by convert_data.py
└── logos/
    ├── teams/               ← your 368 team logo PNGs
    └── conferences/         ← your 32 conference logo PNGs
```

### Step 1 — Generate the data file
Run `convert_data.py` in Python once:
```
python convert_data.py
```
This creates `data/cbb_data.json` from your CSV.

### Step 2 — Set up logos
Copy your logos folders into cbb_site/:
- `cbb_site/logos/teams/`  ← all 368 team PNGs
- `cbb_site/logos/conferences/` ← all 32 conference PNGs

### Step 3 — Run the site
Because the site loads a local JSON file, you need a simple local server.
Open a terminal in the cbb_site/ folder and run:
```
python -m http.server 8000
```
Then open your browser to: http://localhost:8000

### Notes
- Re-run convert_data.py any time you update the CSV
- The site defaults to Villanova 2026 on load
- All percentile views (D1 / Conference / Power vs Mid-Major) work via the toggle on the Profile tab
- Historical trend years default to current year - 1 through current year, adjustable via dropdowns
